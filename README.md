# 🕵️‍♀️ CyberBattle Lab — Attacker vs Defender Simulator

CyberBattle Lab is an interactive cybersecurity simulation platform that allows users to experience both sides of a cyber attack — as an attacker and as a defender.

The project is designed to make cybersecurity concepts practical, visual, and engaging through real-world inspired scenarios.

---

## 🌐 Live Demo

👉 https://falak8484.github.io/cyneros-simulator/

## 🚀 Features

- 🎮 Dual Mode System:
  - **Attacker Mode** — simulate cyber attacks
  - **Defender Mode** — protect systems and respond to threats  

- 🧪 Realistic Cyber Scenarios:
  - Brute-force login attempts  
  - Suspicious IP activity  
  - Malware alerts  
  - Phishing attempts  

- 🖥️ Interactive UI:
  - System-like interface  
  - Live logs and alerts  
  - Action-based decision making  

- 🧠 Learning-Focused Design:
  - Understand attacker mindset  
  - Practice defensive strategies  
  - Visualize cyber incidents  

---

## 🧠 Project Concept

CyberBattle Lab is built to simulate real cybersecurity environments where:

- Attackers attempt to exploit vulnerabilities  
- Defenders monitor, detect, and respond to threats  

This dual-perspective approach helps users understand both:
- how attacks happen  
- how systems can be secured  

---

## 🕹️ Game Modes

### 🔴 Attacker Mode

- Simulate attacks on a target system  
- Perform actions like:
  - Password guessing  
  - Exploit attempts  
  - Network probing  

Goal:
👉 Identify vulnerabilities and gain access  

---

### 🔵 Defender Mode

- Monitor incoming activity  
- Analyze logs and alerts  
- Take actions like:
  - Block IPs  
  - Enable firewall  
  - Investigate suspicious behavior  

Goal:
👉 Protect system and stop attacks  

---

## 🧪 Example Scenarios

- Multiple failed login attempts from same IP  
- Suspicious API requests  
- Unusual traffic spikes  
- Malicious script execution
